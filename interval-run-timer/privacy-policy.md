**SmartApp Studio Lab** (hereinafter "developer") provides the app **Interval Run Timer** (also known on the store as **Interval Run Timer - Floating**, hereinafter "the app" or "the service"). This Privacy Policy explains how the developer handles user data.

**SmartApp Studio Lab**(이하 '개발자')이 제공하는 **인터벌 런 타이머**(스토어 표시명: **인터벌 런 타이머 - 플로팅, 터치 잠금**, 이하 '본 앱' 또는 '서비스')의 개인정보처리방침입니다.

## 1. Data Collection and Use / 수집 및 이용

The app does not directly collect, store, or transmit personal information to the developer's own servers.

본 앱은 개발자 자체 서버로 개인정보를 직접 수집, 저장 또는 전송하지 않습니다.

- Workout data and saved routines are stored locally on the user's device only.
- The app does not directly collect a user's real name, phone number, email address, or precise location.

- 운동 기록과 저장된 루틴은 사용자 기기 내부에만 로컬로 저장됩니다.
- 본 앱은 사용자의 실명, 전화번호, 이메일 주소, 정밀 위치정보를 직접 수집하지 않습니다.

## 3. Permissions Used / 앱 권한 사용

The app may request the following permissions to provide core features. These permissions are used for functionality and are not themselves proof that the developer directly collects personal data.

본 앱은 핵심 기능 제공을 위해 다음 권한을 요청할 수 있습니다. 이 권한들은 기능 제공을 위한 것이며, 그 자체로 개발자가 개인정보를 직접 수집한다는 뜻은 아닙니다.

- 다른 앱 위에 표시(오버레이): 운동 중 진행 상황 표시 및 터치 잠금 기능 제공에 사용

## 4. Data Retention and Deletion / 보관 및 삭제

Because routine and workout data are stored locally on the device, users can remove that data by clearing app data or uninstalling the app. The developer does not keep user data on a separate server.

루틴 및 운동 데이터는 기기에 로컬 저장되므로, 사용자는 앱 데이터 삭제 또는 앱 삭제를 통해 해당 데이터를 제거할 수 있습니다. 개발자는 별도의 서버에 사용자 데이터를 보관하지 않습니다.
